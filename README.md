# Quiz-Management-System
using C++ with STL

COMMANDS
> g++ -c main2.cpp Quiz.cpp
> g++ main2.o Quiz.o -o runApplication
> runApplication

System works
    1. Create C, C++, DSA quiz.
    2. set quiz questions and its options and let the system know the correct answer.
    3. starts the quiz for which the user selects option.
    4. display each question one by one and ask user to answer that question.
    5. calculate score by analysing user answers.
    6. shows result (score)
    7. allow to erase the file data for all files.
